■ What's this?

   プログラムのデバッグを支援するために設計された、開発者向けの ChatGPT API クライアントです。
   使用するには、appsettings.jsonファイルに API キーを入力する必要があります。
   Windows7以降対応のはずですが、Windows10でチェックしています。
   
   [特徴]
 ・縦5分割のメインエディター、プレビュー画面、チャット画面、チャットリスト画面
 ・各種プログラミング言語のシンタックスハイライト対応
 ・ブラウザを内蔵し、Webサービス版のチャットログもインポートできます。（Fileメニューから選択）
 ・20個の定型句ボタン&プリセット（テキストファイルに保存）
 ・入力テキストとチャットの自動ログ記録
 ・会話履歴の自動要約（履歴が長くなった時、自動的に過去の会話を圧縮します。半永久的に会話可能）
 ・自動的にチャットタイトル名を生成
 ・チャットごとにタグを3つまで付けて管理
 ・データベースを使用した全文検索
 ・ログのcsv書き出しとインポート
 ・各種APIパラメータの設定（システムロールでの発言は実装していませんが、パラメータは全て指定可能です）

■ How to

   インストールは不要です。TmCGPTD.exeを起動してください。
   操作に迷うところは無いと思いますが、キーボードショートカットの解説だけします。
   Ctrl + テンキー1～9、0が定型句1～10番に対応、
   Alt + テンキー1～9、0が定型句11～20番に対応しています。
   文字列のテキストファイル保存には、Ctrl+Sを押します。
   Ctrl+Enterでチャット文を送信します。
   また、Ctrl + マウスホイールくるくるでフォントの拡大縮小が出来ます。
